Использовал команды в Git:
1. git clone https://github.com/SergeiM89/vending_machine.git
Создал локальную копию удаленного репозитория, который форкнул ранее.
2. git remote -v
Проверил что все нормально создалось и что у нас в origin (какой удаленный репо).
3. git branch -a
Проверил какие ветки есть локально и на удаленном репо.
4. git checkout home_work_25
Перешел в home_work_25 ветку для работы.
5. git add .
Добавил изменения, которые сделал в индекс.
6. git commit -m "%message%"
Коммитнул изменения в локальную ветку и написал краткое сообщение о том что сделал.
7. git push origin home_work_25
Отправил все коммиты на удаленный репо в ветку home_work_25
8. git checkout master
Перешел в локальную ветку master
9. git pull origin master
Стянул последнюю версию мастера, глянул, что в ней, чтобы было меньше конфликтов (у меня не изменилась т.к никто другой не работал с ней).
10. git merge home_work_25
Смержил home_work_25 ветку в ветку master. Решил возникшие конфликты через интерфейс.
11. git add.
12  git commit -m "Merged home_work_25 в master."
Решенный конфликт снова сохраняю в коммит.
13. git push origin master
Пушу мастер в удаленный репо.
